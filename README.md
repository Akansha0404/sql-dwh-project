# sql-dwh-project
An end-to-end batch data pipeline to process large-scale retail sales data, leveraging Amazon S3 for raw data ingestion and Databricks with PySpark for data transformation and aggregation. Designed dimensional data models (Fact & Dimension tables) and implemented data cleaning, deduplication, and business logic transformations. Performed SQL-based analytics to generate KPIs such as revenue trends, customer segmentation, and product performance insights. Optimized Spark jobs for performance using partitioning and caching techniques.

Tech Stack :
Cloud: AWS S3
Data Processing: Apache Spark, PySpark
Data Modeling: Star Schema (Fact & Dimension Tables)
Analytics: SQL (Spark SQL / Databricks SQL)
Workflow Environment: Databricks Notebook
